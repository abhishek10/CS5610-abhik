Worked on java scripts

This week I worked on java scripts and worked on different types of events for which we can use java scripts.
I worked on events like button click, on load, when click on link and other such events. Using javascript
we can peroform all kind of client side validations on user inputed data before sending it to the server,
reducing chances of sending incorrect data to server. It saves lot of network resources and time.
I performed different experiments to learn this things like -
1. Bouncing ball on button click event
2. Celsius and Fahrenheit conversion
3. Automatic clock on pageload event and other things
